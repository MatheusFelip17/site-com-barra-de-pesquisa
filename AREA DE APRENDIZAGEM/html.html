HTML é a linguagem usada para criar e estruturar páginas da web, organizando textos, imagens e links por meio de tags que indicam como o conteúdo deve ser exibido no navegador.usamos durante o desenvolvimento do projeto, tags semânticas como "div" e "nav", tags de texto como "h1" e a tag "img", além dos atributos identificadores "class" e "id".

"div": é um contêiner genérico usado para agrupar outros elementos e organizar o layout da página, sem ter um significado semântico específico.

"nav": é uma tag semântica que representa uma seção de navegação, geralmente utilizada para conter menus ou links que levam a outras partes do site.
